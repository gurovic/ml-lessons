# Визуализация

Урок «Визуализации для машинного обучения» — 34 слайда + «Источники и практика».

- `plan.md` — план занятия (источник истины)
- `slides_json/` — JSON-слайды (34 + references)
- `assets/generate_visuals.py` — генерация PNG-иллюстраций
- `presentation.pptx` — презентация для проверки
- `code.ipynb` — короткие примеры по слайдам
- `project.ipynb` — мини-проект EDA → модель → диагностика (Titanic)

Перегенерация с нуля:

```bash
python agents/regenerate_vizualizatsiya.py
```
